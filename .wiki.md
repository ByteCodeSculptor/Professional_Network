# Project Summary
The AI-Powered Professional Network is a full-stack application designed to connect professionals and companies, facilitating project postings, applications, and secure messaging. This platform includes features such as user authentication, role-based access control, and payment processing, all while ensuring compliance with data protection regulations.

# Project Module Description
- **Backend**: Built with Node.js, Express, and TypeScript, it includes a REST API with over 40 endpoints, PostgreSQL database integration via Prisma ORM, JWT authentication, role-based access control, and Stripe payment integration.
- **Frontend**: Developed with React, TypeScript, and Tailwind CSS, featuring user registration, dashboards, project management, and a responsive design.
- **Infrastructure**: Utilizes Docker for containerization, with a comprehensive deployment guide for AWS.

# File Description Inventory
- **Dockerfile**: Configuration for containerizing backend and frontend services.
- **package.json**: Manages project dependencies for backend and frontend.
- **schema.prisma**: Defines the database schema using Prisma ORM.
- **server.ts**: Main entry point for the backend server.
- **App.tsx**: Root component for the React frontend application.
- **deployment.md**: Instructions for deploying the application to production.

# Technology Stack
- **Backend**: Node.js, Express, TypeScript, PostgreSQL, Prisma, JWT, Redis, Stripe
- **Frontend**: React, TypeScript, Tailwind CSS, Shadcn-ui
- **Infrastructure**: Docker, AWS

# Usage
1. **Install Dependencies**: Run `npm install` in both the backend and frontend directories.
2. **Configure Environment Variables**: Refer to `.env.example` for setup.
3. **Run Locally**: Use Docker with `docker-compose up` or manually start the backend and frontend servers.
4. **Deploy to Production**: Follow the deployment guide in `docs/deployment.md`.
